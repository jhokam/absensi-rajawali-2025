## Run Locally

1. Get the code

```bash
  git clone https://github.com/jhokam/absensi-rajawali-2025
```

2. Install package

```bash
# bun
bun install

# npm
npm install
```

3. Run the code

```bash
# bun
bun dev

# npm
npm run dev
```
